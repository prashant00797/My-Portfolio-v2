## Projects

### Fullstack AI

**Viola – Portfolio Assistant**
An agentic AI chat widget powered by LangGraph, RAG and FastAPI. Answers questiosn with respect to the knowledge base, fetches live GitHub repos, captures visitor details and streams responses token by token via SSE.

- Frontend: `same-repo check components/chat folder`
- Backend: [personal-chat-agent](https://github.com/prashant00797/personal-chat-agent)
- Stack: LangGraph · FastAPI · Pinecone · Supabase · OpenAI · Next.js

  Add environment variable:

```bash
NEXT_PUBLIC_API_URL=http://localhost:8000/api/chat
```
